# SO100 Demo

This folder contains demo files and examples for the SO100 system integration.

## Purpose

Demonstrates the functionality and usage of the HFSO100 to AnninAR4 connector with sample SO100 data, including visualisation mode based on the leader arm and cartesian-based control of SO100.

## Contents

- Sample configuration files
- Test data examples
- Usage demonstrations
- Visualisation mode examples using leader arm data
- Cartesian-based control scripts for SO100

## Usage

To get started with the demo:

1. **Setup**: Copy the sample configuration files to your project directory and adjust parameters as needed for your environment.

2. **Running Demonstrations**: Execute the usage demonstration scripts to see basic integration examples. For instance, run `demo_script.py` (or equivalent) in a Python environment with required dependencies installed.

3. **Visualisation Mode**: Use the visualisation examples to load leader arm data. Run the visualisation script, e.g., `visualize_leader_arm.py`, to render 3D models or plots of the arm movements. Ensure you have a compatible viewer or library like Matplotlib or Open3D.

4. **Cartesian Control**: Test cartesian-based control by running the provided scripts, such as `cartesian_control.py`. This will simulate or execute movements of the SO100 system in cartesian coordinates. Input sample data from the test data examples for validation.

5. **Testing**: Use the test data examples to validate outputs. Compare results against expected behaviors documented in the main project README.

Refer to the main project README for detailed setup and execution instructions, including dependency installation and troubleshooting.
